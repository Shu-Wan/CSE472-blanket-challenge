# Evaluations & Rubrics

## Submissions

Create a fork from `main` and submit a PR before the due date (Nov 24, EOD AZ time).
The last commit before the deadline will be evaluated.

- This is a team project, so one PR per team is sufficient.
- (Optional) Lab and bonus points can be submitted individually. Please let me
  know if you want to be evaluated individually.
- If there are multiple PRs from the same team, the latest PR before the due
  date will be evaluated.
- Everything except the final report should be submitted via PR.
- For report instructions, see: [REPORT.md](/docs/REPORT.md)

## Rubrics

### Labs

- Code does not run or is largely incomplete: min 0 points
- Reproduce the provided results: 4 points
- Correctly answer the questions: 1 point
- Each lab includes up to 5 bonus points.

### Final Project

- Implementation
  - Code does not run or is largely incomplete: min 0 points
  - Reproduce the baseline results: 25 points
  - Model that outperforms the baseline: 15 points
  - Bonus: Insightful analysis of the results: up to 5 points
- Results
  - Based on the final project ranking
  - 1st place: 30 points; last place: 10 points
  - Rest: [min-max scaled](https://en.wikipedia.org/wiki/Feature_scaling#:~:text=Rescaling%20(min%2Dmax%20normalization))
    between 10 and 30 points
  - For ranking details, see [Final Project Instructions](/labs/04_final/README.md).
